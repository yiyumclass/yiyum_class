## 핵심 결정 사항 (요약)

- **가입과 결제 분리**: 가입은 무료, 강의·전자책마다 개별 결제 → 이용권(entitlement) 부여.
- **DB/인증**: Supabase (Auth + Postgres + RLS).
- **결제**: 토스페이먼츠 일회성 결제. 공개 테스트 키로 계정 없이 전 구간 테스트 가능.
- **인증 방식**: 카카오 로그인 우선 (비즈앱 전환 대기), 필요 시 이메일+비번 대안.
- **로그인 게이트**: 랜딩은 오픈, "결제하려는 순간"에만 로그인 요구 (로그인 후 원래 흐름 복귀).
